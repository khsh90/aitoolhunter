## ğŸ® How to Use

### Method 1: Auto-Generate (Recommended)

1. **Enter tool name**: Type "Notion", "ChatGPT", etc.
2. **Click "Auto-Generate All"**: Wait 10-20 seconds
3. **Extended fields appear automatically** if tool is on Futurepedia
4. **Edit if needed**: Add/remove/modify items
5. **Click "Save Tool"**

### Method 2: Manual Entry

1. **Fill basic fields**: Name, category, etc.
2. **Click "Show Enhanced Data Fields"** button
3. **Purple section appears**
4. **Click "+ Add Feature"**: Type a feature
5. **Click "+ Add Feature"** again: Add more features
6. **Repeat for Pros and Cons**
7. **Fill "What Makes It Unique"** text area
8. **Click "Save Tool"**

### Method 3: Edit Existing Tool

1. **Click Edit icon** (âœï¸) on any tool in the list
2. **Form fills with existing data**
3. **Extended fields appear automatically** if tool has them
4. **Edit anything you want**
5. **Click "Update Tool"**

## ğŸ› Troubleshooting UI Issues

### "I don't see the button"

**Check:**
- Are you in "Add New Tool" mode? (not editing)
- Is the extended section already open?
- Have you already added items?

**Solution:** The button only shows when section is closed AND empty.

### "The section won't close"

**Why:** You have data in it (features, pros, cons, etc.)

**Solution:** Remove all items first, then close button appears.

### "After auto-generate, I don't see extended fields"

**Why:** Tool not found on Futurepedia (normal for unknown tools)

**Solution:** Click "Show Enhanced Data Fields" to add manually.

### "Extended fields are empty after save"

**Check:**
1. Browser console for errors
2. Terminal for save payload logs
3. Appwrite field configuration

**Solution:** See [APPWRITE_TROUBLESHOOTING.md](APPWRITE_TROUBLESHOOTING.md)
